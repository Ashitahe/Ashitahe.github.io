# 添加 bili 插件

本插件是由 hexo 原生 youtube 标签插件更改而来的，用相同的方法可以添加各大视频网站的标签。

# 具体步骤：

  • 找到 hexo 安装目录下的标签插件存放位置
  • 创建一个 bili.js 文件，将代码写入
  • 注册该标签插件
  • 使用

# 使用

直接根据语法使用即可，建议使用 b 站官方的分享链接，不然打开文章时可能会直接跳转到 b 站，复制 src 里的内容

效果

# 缺点

只能以默认 360p 的分辨率播放视频，若想切换分辨率则会自动跳转到 b 站。
