# Selenium 基本使用

# 简介

Selenium 是一种驱动浏览器的库，它通过调用 Webdriver 来驱动电脑中的浏览器从而使我们可以获得一个真实的浏览器环境而不是模拟浏览器环境获取网页数据。它提供了一系列 API 来帮助我们获取浏览器中的内容。

使用它之前需要下载对应浏览器版本的 Webdriver ：

  • Chrome Webdriver
  • Edge Webdriver
  • FireFox Webdriver
  • Safari Webdriver

在合适的目录中放置好 Webdriver 后我们就可以通过 Selenium 调用它，从而获取网页内容。
