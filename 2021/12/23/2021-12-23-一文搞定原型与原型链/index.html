# 函数对象的原型链

到目前为止，我们基本搞定了普通对象的原型链结构。但是还有一个问题我们没有搞定就是，既然函数也是一个对象那么它肯定也有自己的隐式原型 __proto__ 指向它的构造函数的原型。在 js 的底层所有的函数都是由 Function 作为构造函数 new 出来的，也就是说任何一个函数都是 Function 的实例
